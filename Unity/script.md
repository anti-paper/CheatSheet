# スクリプト（C#）
スクリプトの書き方をある程度体系的にまとめるつもり

## 一覧
C#標準搭載のものは除く（Unity独自のコードのみメモ）<br>
C#標準搭載のものはC#のメモファイルに記載<br>
<br><br>

|コード|機能|備考|
|--|--|--|
|`Debug.Log(文字列)`|文字列のログ出力|-|
|`Vector2 変数名 = new Vector2(X座標, Y座標)`|指定した座標を「(X座標, Y座標)」の形式で持つように宣言|メンバ変数x, yにアクセスすることで読み取り、更新可能<br>Vector2型同士の加減算も可能|
|`Input.GetMouseButtonDown(整数)`|マウスがクリックされた瞬間に1回だけtrueになる|引数「0」→左クリック<br>引数「1」→右クリック<br>引数「2」→中ボタンクリック|
|`Input.GetMouseButton(整数)`|マウスがクリックされている間ずっとtrueになる|引数「0」→左クリック<br>引数「1」→右クリック<br>引数「2」→中ボタンクリック|
|`Input.GetMouseButtonUp(整数)`|マウスボタンが離された瞬間に1回だけtrueになる|引数「0」→左クリック<br>引数「1」→右クリック<br>引数「2」→中ボタンクリック|
|`ゲームオブジェクト.transform.Rotate(X軸回転, Y軸回転, Z軸回転)`|指定したパラメータに従ってオブジェクトが回転|スクリプトがアタッチされたオブジェクトを回転させる場合はゲームオブジェクトの指定を省略可|
|`Input.mousePosition`|マウスの座標|メンバ変数x, yを持つ|
|`ゲームオブジェクト.transform.position`|ゲームオブジェクトの座標を取得|メンバ変数x, yを持つ<br>スクリプトがアタッチされたオブジェクトの座標を取得する場合はゲームオブジェクトの指定を省略可|
|`ゲームオブジェクト.transform.Translate(X軸方向移動距離, Y軸方向移動距離, Z軸方向移動距離)`|引数で指定した距離だけゲームオブジェクトを移動|ローカル座標系での指定になる<br>スクリプトがアタッチされたオブジェクトを移動させる場合はゲームオブジェクトの指定を省略可|
|`Input.GetKeyDown(KeyCode.押されたキーを示す定数)`|押されたキーを認識する|キーごとの定数は以下の通り（随意更新予定）<br>↑ : UpArrow<br>↓ : DownArrow<br>→ : RightArrow<br>← : LeftArrow<br>各種アルファベットキー : 大文字アルファベット|
|`Destroy(ゲームオブジェクト)`|指定したゲームオブジェクトを削除|引数を「gameObject」にすると、スクリプトをアタッチしたゲームオブジェクトが削除される|
|`Time.deltaTime`|前フレームと現在のフレームの時間差|-|
|`Instantiate(プレハブ)`|引数に指定したプレハブのインスタンスを作成する|戻り値の方はObject型|
|`値 as 型`|値の型を指定した型にキャスト（強制型変換）|-|
|`Random.Range(開始値, 終了値)`|開始値以上終了値未満の範囲でランダムな整数を生成|-|
|`GameObject.Find(オブジェクト名)`|名前でゲームオブジェクトを特定|名前が重複してる場合は特定の仕方が面倒になる＆オブジェクトの数によっては探すのに時間がかかるので使う場面は限られる|
|`ゲームオブジェクト.GetComponent<コンポーネント名>()`|指定したゲームオブジェクトの指定したコンポーネントを取得|「.」で繋げて各種パラメータにアクセス可能|
|`[SerializeField] 型 変数名`|指定した型の変数を宣言し、インスペクタ上に表示させる|インスペクタ上で編集可能であり、文字列、数値の場合は直接入力、オブジェクトの場合はアタッチする|
<br><br>

### ワールド座標系とローカル座標系
|座標系|意味|
|--|--|
|ワールド座標系|絶対的な座標<br>ユーザ目線の座標|
|ローカル座標系|相対的な座標<br>オブジェクト目線の座標|

### あとでC#メモに清書
ToString<br>
引数：<br>
- D5→5桁の整数（0埋め）<br>
- F3→小数点以下3桁まで表示した小数（0埋め）<br>
即値をToStringする場合は即値を()で囲む<br>
例：`(123).ToString("D5")`
