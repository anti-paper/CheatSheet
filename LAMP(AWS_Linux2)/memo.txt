ec2→linux2,新規セキュリティグループ
セキュリティグループ→http(80),https(443)追加→ssh(22)削除
IAMロール→「ssmmanaged」検索→アタッチ→インスタンス再起動
ソフトウェアアップデート→コマンド「sudo yum -y update」実行
Apacheインストール→コマンド「sudo yum -y install httpd」実行
Apache起動→コマンド「sudo systemctl start httpd」実行
インスタンス起動時自動でApache起動→コマンド「sudo systemctl enable httpd」実行
Apache起動状況確認→コマンド「sudo systemctl is-enabled httpd」実行
公開ファイル置き場→/var/www/html/
ローカルタイムの設定→コマンド「sudo ln -sf /usr/share/zoneinfo/Japan /etc/localtime」実行
PHPインストール→コマンド「sudo yum -y install php」実行
PHP設定ファイルバックアップ→コマンド「sudo cp /etc/php.ini /etc/php.bak」実行
Apache再起動→コマンド「sudo systemctl restart httpd」実行
PHP設定変更→コマンド「sudo vim /etc/php.ini」実行（Vim起動→php.ini修正）
→「error_reporting」に「& ~E_NOTICE」追加→「display_errors」を「Off」から「On」へ変更
MySQLインストール→コマンド「sudo yum -y install mysql-server」実行
PHPのMySQLネイティブドライバインストール→コマンド「sudo yum -y install php-mysqlnd」実行
MySQL起動→コマンド「sudo systemctl start mysqld」実行
インスタンス起動時自動でMySQL起動→コマンド「sudo systemctl enable mysqld」実行
MySQL起動状況確認→コマンド「sudo systemctl is-enabled mysqld」実行
MySQL設定→コマンド「sudo mysql-secure-installation」実行
MySQL文字コード設定変更→コマンド「sudo vim /etc/my.cnf」実行（Vim起動→my.cnf修正）
→「character-set-server」を「utf8」へ変更
MySQL再起動→コマンド「sudo systemctl restart mysqld」実行
phpMyAdminのダウンロード先追加→コマンド「sudo yum-config-manager --enable epel」実行
phpMyAdminインストール→コマンド「sudo yum -y install phpmyadmin」実行
phpMyAdminアクセス制限設定変更→コマンド「sudo vim /etc/httpd/conf.d/phpMyAdmin.conf」実行（Vim起動→phpMyAdmin.conf修正）
→「127.0.0.1」をwebサーバのグローバルIPアドレスへ変更
Apache再起動→コマンド「sudo systemctl restart httpd」実行
