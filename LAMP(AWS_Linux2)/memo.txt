ec2→linux2,新規セキュリティグループ
セキュリティグループ→http(80),https(443)追加→ssh(22)削除
IAMロール→「ssmmanaged」検索→アタッチ→インスタンス再起動
ソフトウェアアップデート→コマンド「sudo yum -y update」実行
Apacheインストール→コマンド「sudo yum -y install httpd」実行
Apache起動→コマンド「sudo systemctl start httpd」実行
インスタンス起動時自動でApache起動→コマンド「sudo systemctl enable httpd」実行
Apache起動状況確認→コマンド「sudo systemctl is-enabled httpd」実行
公開ファイル置き場→/var/www/html/
ローカルタイムの設定→コマンド「sudo ln -sf /usr/share/zoneinfo/Japan /etc/localtime」実行
PHPインストール→コマンド「sudo yum -y install php」実行
PHP設定ファイルバックアップ→コマンド「sudo cp /etc/php.ini /etc/php.bak」実行
Apache再起動→コマンド「sudo systemctl restart httpd」実行
PHP設定変更→コマンド「sudo vim /etc/php.ini」実行（Vim起動→php.ini修正）
→「error_reporting」に「& ~E_NOTICE」追加→「display_errors」を「Off」から「On」へ変更
